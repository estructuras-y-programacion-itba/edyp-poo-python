# Programación Orientada a Objetos en Python

Material de referencia para la materia **Estructuras de Datos y Programación** — ITBA.

Está orientado a estudiantes de carreras no informáticas que ya programan en Python y quieren incorporar el paradigma orientado a objetos.

## Prerequisitos

- Python 3.10 o superior
- Conocimientos básicos de Python: variables, funciones, `if`, `for`, listas y diccionarios
- Sin experiencia previa en POO requerida
